Maze

Personal
July 2022
GitHub
Maze is a little mini-game that runs within the command line. Powered by Ruby, it creates mazes of random textures of any sizes using randomized versions of graph traversal algorithms. For example, when using Randomized Depth-First Search, mazes will consist of long corridors due to its nature of traversing as deep as possible. On the other hand, using Randomized Kruskal's algorithm will create more unique paths as the algorithms always generate a random spanning tree.
Game Logic, Command Line Control, Documentation (using YARD)

MicroCaml

#lexer #parser #evaluator #CFGs #functionalprog
Academic
June 2022
No Resources Available
MicroCaml was an academic project focusing on the study of Context-Free Grammars and code compilation. It is intended to be a "mini" version of OCaml itself, having the same syntax albeit less functionalities. To implement this, MicroCaml code has to be tokenized into a series of tokens, which the parser can use to generate an Abstract Syntax Tree for evaluation. Any logical or syntax errors will be caught in this final step. It was a fun little project that helped introduced the concept of functional programming. (Due to potential risk of academic violation, the source code will not be publicly shared)
Lexer, Parser, and Evaluator Implementation

Caraoke is a submission to the Bitcamp 2022 Hackathon hosted at the University of Maryland. The vision of this project is to provide a portable lyrics display machine to fit within the A/C vents of a vehicle. The lyrics would sync with whatever song is playing on your mobile device, making singing along with friends and family on long trips even more fun. For this project, we implemented a connection with Spotify's Developer API, grabbed the lyrics from Textyl.co API, and sent it over to our Arduino Genuino 101 board that handles the text being displayed on the LCD Screen. It was a great team-building and collaboration experience.
Handling the API, Data Processing
Microcontroller, Circuit Board, LCD Screen
